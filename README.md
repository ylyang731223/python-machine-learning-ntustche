## 課程簡介

本課程從 Python 程式設計基礎出發，結合數值方法與機器學習，應用於化學工程問題。課程以互動式網頁教學為核心，每個單元包含語法教學與化工數值方法習題，讓學生在學習程式語言的同時，建立解決工程問題的計算思維。

## 課程大綱

| 週次 | 主題 | 內容 |
|:----:|------|------|
| 1–2 | Python 入門與基本語法 | 變數、資料型態、算術運算 |
| 3–4 | 內建資料結構 | 字串、List、Tuple、Dict |
| 5–6 | 流程控制 | 比較／邏輯運算子、if-else、條件運算式 |
| 7–8 | 迴圈與迭代 | for / while、break / continue、巢狀迴圈 |
| 9–10 | Pandas、NumPy、Matplotlib | 資料處理、陣列運算、視覺化 |
| 11–12 | 函式與最佳化 | Comprehension、def / closure / lambda、SciPy optimize |
| 13 | 機器學習基礎 | 梯度下降、線性迴歸、PyTorch 入門 |
| 14 | 監督式學習 | 分類與迴歸、Scikit-Learn 實作 |
| 15 | 深度學習與神經網路 | PyTorch 建模、訓練流程 |
| 16 | 生成式 AI 與化工應用 | Transformer 基礎、化工案例研討 |

## 教學特色

- **互動式網頁教學**：純 HTML，零外部依賴，支援離線使用與深色／淺色主題
- **46 道化工習題**：每題搭配 SVG 向量示意圖、背景知識、核心概念與化工意涵
- **逐步引導模式**：每道習題提供分步驟引導，學生可逐步展開提示
- **數值方法實作**：從基礎的 Taylor 展開到進階的 ODE 求解與最佳化，皆以化工實際問題為載體
- **銜接機器學習**：最後四週由數值方法自然過渡至梯度下降、PyTorch 與深度學習
