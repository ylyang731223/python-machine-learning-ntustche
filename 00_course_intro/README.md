## 內容架構

### Unit 2：教科書

| 書名 | 用途 |
|------|------|
| *Basic Principles and Calculations in Chemical Engineering* (Himmelblau) | 化工原理 |
| *Applied Mathematical Methods for Chemical Engineers* (Loney) | 數值方法 |
| *Machine Learning with PyTorch and Scikit-Learn* (Raschka) | 機器學習 |
| *Hands-On Generative AI with Transformers and Diffusion Models* | 生成式 AI |

### Unit 3：課程大綱

| 週次 | 主題 | 內容 |
|:----:|------|------|
| 1–2 | Python 入門與基本語法 | 變數、資料型態、算術運算 |
| 3–4 | 內建資料結構 | 字串、List、Tuple、Dict |
| 5–6 | 流程控制 | 比較／邏輯運算子、if-else、條件運算式 |
| 7–8 | 迴圈與迭代 | for / while、break / continue、巢狀迴圈 |
| 9–10 | Pandas、NumPy、Matplotlib | 資料處理、陣列運算、視覺化 |
| 11–12 | 函式與最佳化 | Comprehension、def / closure / lambda、SciPy optimize |
| 13 | 機器學習基礎 | 梯度下降、線性迴歸、PyTorch 入門 |
| 14 | 監督式學習 | 分類與迴歸、Scikit-Learn 實作 |
| 15 | 深度學習與神經網路 | PyTorch 建模、訓練流程 |
| 16 | 生成式 AI 與化工應用 | Transformer 基礎、化工案例研討 |

### Unit 4：期末報告格式

- 1–4 人一組，自行組隊
- 報告題目：2025 年後發表且與化工／化學／材料 AI 應用相關之 SCI 期刊文獻，並附有 GitHub 程式碼
- 繳交：投影片（.pptx）＋ 影片（.mp4）
- 評分標準：口條流暢度、投影片格式、內容完整性、主題闡述、分析討論、創新獨特性

### Unit 5：課程工具介紹

課程使用的軟體工具，包含 Python 與 AI 輔助開發工具。
