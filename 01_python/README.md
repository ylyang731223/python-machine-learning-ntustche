### 核心概念補充

- **Ex 1-1**：數值不穩定性（Numerical Instability）— 項的大小演變（上升→峰值→下降）、災難性抵消、恆等式 exp(−x) = 1/exp(x) 與區間拆分（Range Reduction）策略
- **Ex 1-2**：IEEE 754 雙精度儲存結構（1+11+52 位元）、為何只有 15–16 位有效數字、災難性抵消的成因
- **Ex 1-3**：O(h) 大 O 記號定義、Taylor 展開推導前向差分 O(h) 與中央差分 O(h²) 精度差異
- **Ex 1-6**：誤差傳播的 Taylor 展開推導、偏導數與各參數貢獻百分比分析
