    字串索引與切片

    學習如何透過索引和切片來存取字串中的字元。

    索引 (Indexing)

    s = "Python"
#     P  y  t  h  o  n
#     0  1  2  3  4  5   ← 正向索引
#    -6 -5 -4 -3 -2 -1   ← 反向索引

print(s[0])     # P
print(s[-1])    # n（最後一個字元）

    切片 (Slicing)

    💡
    語法：s[start:end:step] — start 包含，end 不包含，step 為步長。
    s = "Python"
print(s[0:3])    # Pyt（索引 0, 1, 2）
print(s[:3])     # Pyt（從頭開始）
print(s[3:])     # hon（到結尾）
print(s[::2])    # Pto（步長 2）
print(s[::-1])   # nohtyP（反轉字串）

    字串不可變 (Immutable)

    s = "Python"
s[0] = "J"   # TypeError! 字串不可修改
s = "J" + s[1:]  # 只能重新賦值 → "Jython"

    負數索引與 len()

    a = 'NTUCHE'
print(len(a))     # 6（字串長度）
print(a[-2])      # H（倒數第二個）
print(a[0:6:2])   # NUH（步長 2）
print(a[::-2])    # EHT（反向步長 2）

# 切片超出範圍不會報錯（索引超出才會）
print(a[3:42])    # CHE（自動截斷）

    List 基礎

    List（串列）是 Python 最常用的資料結構，有序且可變。

    # 建立 List
fruits = ["apple", "banana", "cherry"]

    List 方法

    掌握 List 常用的增刪改查方法。

    方法功能說明
    append(x)末尾新增在 List 末尾加入元素 x
    insert(i, x)指定位置插入在索引 i 處插入元素 x
    pop(i)移除並回傳移除索引 i 的元素（預設最後一個）
    del list[i]刪除元素刪除索引 i 的元素
    sort()排序原地排序（由小到大）
    reverse()反轉原地反轉順序
    lst = [3, 1, 4, 1, 5]
lst.append(9)        # [3, 1, 4, 1, 5, 9]
lst.insert(0, 2)     # [2, 3, 1, 4, 1, 5, 9]
lst.pop()             # 移除 9 → [2, 3, 1, 4, 1, 5]
lst.pop(0)            # 移除 2 → [3, 1, 4, 1, 5]
del lst[2]           # 刪除索引2 → [3, 1, 1, 5]
lst.sort()            # [1, 1, 3, 5]
lst.reverse()         # [5, 3, 1, 1]
    ⚠
    注意：sort() 和 reverse() 是「原地操作」，回傳值是 None。

    append vs += 差異

    # append() — 整個物件作為一個元素加入
x = [1, 2, 3]
x.append([4, 5])
print(x)   # [1, 2, 3, [4, 5]] ← 巢狀！

# += — 逐一合併元素
y = [1, 2, 3]
y += [4, 5]
print(y)   # [1, 2, 3, 4, 5] ← 展開合併

# pop() 會回傳被移除的值
z = [1, 2, 3]
val = z.pop()
print(val, z)  # 3 [1, 2]

    Tuple 與 Range

    Tuple（元組）是不可變的有序集合；Range 用於產生整數序列。

    Tuple 基礎

    t = (1, 2, 3)
t2 = ("a",)          # 單元素 Tuple 要加逗號！
print(t[0])            # 1
t[0] = 10             # TypeError! 不可變

    Tuple Packing & Unpacking

    # Packing
coords = 3, 4, 5
print(coords)          # (3, 4, 5)
# Unpacking
x, y, z = coords
print(x, y, z)         # 3 4 5

    Range

    print(list(range(5)))         # [0, 1, 2, 3, 4]
print(list(range(2, 6)))      # [2, 3, 4, 5]
print(list(range(0, 10, 2)))  # [0, 2, 4, 6, 8]

    Tuple 易錯點與技巧

    # 小心！(1) 是 int，不是 Tuple
print(type((1)))     # <class 'int'>
print(type((1,)))   # <class 'tuple'> ← 加逗號才是

# 利用 Tuple 交換變數
a, b = 'NTU', 'CHE'
a, b = b, a
print(a, b)          # CHE NTU

# Tuple 內的 list 可以修改
t = (1, 2, [3, 4])
t[2][0] = 99
print(t)             # (1, 2, [99, 4])

    student = {"name": "Alice", "age": 20, "major": "化工"}

    常用方法

    方法功能回傳值
    keys()取得所有鍵dict_keys([...])
    values()取得所有值dict_values([...])
    items()取得所有鍵值對dict_items([...])

    clear()清空字典None

    蒸汽表線性內插

    線性內插法 — 過熱蒸汽焓值查表與內插

    📚
    背景知識：線性內插是工程計算中最常用的數據查表方法。給定兩個已知數據點 (x0, y0) 和 (x1, y1)，在 x0 < x < x1 的任意 x 處，y ≈ y0 + (y1 - y0) * (x - x0) / (x1 - x0)。蒸汽表是化工熱力學的核心工具，經常需要在表列溫度之間進行內插。

    示意圖

    🎯
    核心概念：
    • 線性內插在兩個相鄰數據點之間假設函數呈直線變化，其精度取決於數據間距和函數的曲率——間距越小、曲率越低，線性假設越合理。
    • 內插 (interpolation) 與外插 (extrapolation) 的根本區別在於：內插在已知數據包圍的區域內進行，有數據約束；外插則延伸到數據範圍外，無約束地依賴模型假設，誤差可能急遽增大。
    • list of tuples 結構 [(T₁,H₁), (T₂,H₂), ...] 將自變數與因變數配對儲存，便於透過索引定位區間；在更大規模的應用中可改用字典或二維陣列以提升查詢效率。

    演算法步驟

    ⚙
    1. 將數據儲存為 list of tuples: [(x0,y0), (x1,y1), ...]
    2. 確定查詢點 xq 所在的區間 [xi, xi+1]
    3. 套用線性內插公式：yq = y0 + (y1−y0)·(xq−x0)/(x1−x0)
    4. 若查詢點超出數據範圍，標記為外插並印出警告
    5. 整理並印出所有結果

    題目

    以下是 100 kPa 下過熱蒸汽的焓值數據：

  T (°C)    H (kJ/kg)
  100       2676.2
  150       2776.4
  200       2875.3
  250       2974.3
  300       3074.3

將數據存為 list of tuples：steam_data = [(100, 2676.2), (150, 2776.4), ...]

請在以下溫度進行線性內插：
  1. T = 135°C（在 100-150 之間）
  2. T = 155°C（在 150-200 之間）
  3. T = 190°C（在 150-200 之間）

另外，若查詢 T = 350°C（超出數據範圍），應印出外插警告。

    逐步引導

    Step 1

    建立 steam_data 為 list of tuples，每個 tuple 包含 (T, H)。印出數據表。

    提示：steam_data = [(100, 2676.2), (150, 2776.4), (200, 2875.3), (250, 2974.3), (300, 3074.3)]
    Step 2

    對 Tquery = 135°C，找出包圍的兩個數據點。 用索引存取：steam_data[0] 和 steam_data[1]。 計算線性內插值。

    提示：T0, H0 = steam_data[0]; T1, H1 = steam_data[1] Hinterp = H0 + (H1 − H0) · (Tquery − T0) / (T1 − T0)
    Step 3

    對 Tquery = 155°C 和 190°C 重複同樣的計算，注意選擇正確的區間。

    提示：155 和 190 都在 steam_data[1] 和 steam_data[2] 之間。
    Step 4

    對 Tquery = 350°C，檢查是否超出數據範圍。 如果超出，印出警告訊息，並用最後兩個點進行外插（但標註為外插）。

    提示：比較 Tquery 與 steam_data[-1][0]（最大溫度）。
    Step 5

    整理並印出所有內插結果的摘要表。

    預期輸出

    100 kPa 過熱蒸汽焓值數據
T (°C)    H (kJ/kg)
100       2676.2
150       2776.4
200       2875.3
250       2974.3
300       3074.3

線性內插結果：
T = 135°C → H = 2746.34 kJ/kg (內插，區間 100-150°C)
T = 155°C → H = 2786.29 kJ/kg (內插，區間 150-200°C)
T = 190°C → H = 2855.48 kJ/kg (內插，區間 150-200°C)

⚠ 警告：T = 350°C 超出數據範圍 (100-300°C)，以下為外插結果！
T = 350°C → H = 3174.3 kJ/kg (外插，使用區間 250-300°C)

    Antoine 方程字典與泡點壓力

    字典資料結構、Raoult 定律計算 — 苯-甲苯二成分系統泡點壓力

    📚
    背景知識：Antoine 方程 log10(Psat) = A − B/(T+C) 是計算純物質飽和蒸氣壓的經典公式。Python 的 dict（字典）是儲存物性數據的理想結構，可以用化合物名稱作為 key。泡點壓力由 Raoult 定律計算：Pbubble = Σ xi·Pisat，這是化工熱力學中汽液平衡 (VLE) 的基礎。 【泡點 (Bubble Point) 的物理意義】 泡點是液態混合物開始沸騰（出現第一個氣泡）的條件。 在給定溫度下，泡點壓力是使液體剛好開始汽化的壓力。 對理想溶液，泡點壓力由 Raoult 定律給出： Pbubble = Σ xi · Pisat(T) 其中 xi 是液相莫耳分率，Pisat(T) 是純物質 i 在溫度 T 的飽和蒸氣壓。 物理直覺：當系統總壓等於泡點壓力時，液體表面開始形成氣泡。 總壓高於泡點壓力 → 全部為液態；低於泡點壓力 → 開始汽化。 泡點壓力介於各純物質飽和蒸氣壓之間，隨揮發性較高成分的 莫耳分率增加而上升。

    示意圖

    🎯
    核心概念：
    • Antoine 方程 log₁₀(Psat) = A − B/(T+C) 用三個經驗參數即可精確擬合大多數純物質的飽和蒸氣壓曲線，是 Clausius-Clapeyron 方程的實用近似形式。
    • Raoult 定律 Pbubble = Σxi·Pisat 假設理想溶液行為，泡點壓力與液相組成呈線性關係；圖中 P-x 圖的直線特徵正是理想溶液的標誌，實際非理想系統會偏離此直線。
    • 巢狀字典 (nested dict) 以化合物名稱為外層 key、Antoine 係數為內層 key-value，實現了結構化的物性數據存取；此設計模式可擴展至數百種化合物的物性資料庫。
    • 泡點壓力隨高揮發性成分（苯）莫耳分率增加而線性上升，反映出各成分對混合物總蒸氣壓的加權貢獻。

    5. 整理結果並單位轉換 (mmHg → atm)

    分子量計算器

    字典查表與 tuple 資料結構 — 化學式分子量計算與燃燒反應質量守恆

    📚
    背景知識：Python 的 dict 非常適合儲存元素週期表數據，以元素符號為 key、原子量為 value。化學式可以用 list of tuples 表示，每個 tuple 為 (元素, 個數)。分子量 = Σ(原子量 × 個數)。這也是質量守恆計算的基礎。

    示意圖

    🎯
    核心概念：
    • Python 字典將元素符號映射到原子量，實現了 O(1) 時間複雜度的查表操作；這與化學家使用週期表的直覺一致——知道元素符號就能立即找到原子量。
    • list of tuples 表示化學式 [("C",2), ("H",6), ("O",1)] 清楚地將元素種類與原子個數配對，分子量的計算即為加權求和 MW = Σ(原子量×個數)。

    Lagrange 內插法

    Lagrange 內插多項式 — 甲苯密度數據內插

    📚
    背景知識：Lagrange 內插法可用 n 個數據點建立 n-1 次多項式。3 點 Lagrange 內插使用 2 次多項式，比線性內插（1 次）更精確。公式：P(x) = Σ yi·Li(x)，其中 Li(x) = Π(x−xj)/(xi−xj), j≠i。在化工物性數據查詢中，Lagrange 內插比線性內插能更好地捕捉非線性趨勢。

    示意圖

    🎯
    核心概念：
    • Lagrange 內插用 n 個數據點建構唯一的 n-1 次多項式通過所有數據點；3 點 Lagrange 產生二次多項式，能捕捉數據的曲率，而線性內插（2 點）只能擬合直線變化。
    • Lagrange 基底多項式 Lᵢ(x) 的關鍵性質為：在自身節點 xᵢ 值為 1、在其他節點值為 0，且所有基底之和恆等於 1（ΣLᵢ=1）；這確保了內插值是數據點的加權平均，權重隨距離動態調整。
    • 此例中線性內插與 Lagrange 內插的差異僅 0.04 kg/m³，因甲苯密度隨溫度近乎線性下降；但對於曲率較大的物性數據（如接近臨界點的密度），二次內插的優勢會更加顯著。
    • 當數據點過多（>6~7）且等距分布時，高階 Lagrange 多項式可能在邊緣產生 Runge 現象（劇烈震盪），此時應改用分段低階內插（如三次樣條）。

    演算法步驟

    ⚙
    1. 準備數據點 (xi, yi)，選擇包圍查詢點的 3 個數據點
    2. 計算 Lagrange 基底多項式：L0, L1, L2
    3. L0(x) = (x−x1)(x−x2) / ((x0−x1)(x0−x2))
    4. 計算內插值：P(x) = L0·y0 + L1·y1 + L2·y2
    5. 驗證：L0 + L1 + L2 = 1（必要條件）
    6. 與線性內插結果比較

    熱容多項式與焓計算 (Horner's method)

    Horner 法多項式求值、數值積分 — 氣體熱容計算與氫氣燃燒反應焓

    📚
    背景知識：氣體熱容常表示為溫度的多項式：Cp = a + bT + cT² + dT³。Horner 法（巢狀乘法）可減少乘法次數：Cp = a + T*(b + T*(c + T*d))。焓變化量 ΔH = ∫T₂T₁ Cp dT 可解析積分。將多種氣體的 Cp 係數存為巢狀字典，是化工計算中組織物性數據的好方法。

    示意圖

    🎯
    核心概念：
    • Horner 法將 n 次多項式 a+bT+cT²+dT³ 改寫為巢狀形式 a+T(b+T(c+Td))，將乘法次數從 n(n+1)/2 降至 n 次，同時減少中間結果的數量級跳動，提升數值穩定性。
    • 熱容多項式 Cp(T) 對 T 的積分 ∫Cp dT 可解析求得各項 aT+bT²/2+cT³/3+dT⁴/4 在上下限的差值，這是解析積分與數值積分互相驗證的好例子。
    • 反應焓隨溫度的修正 ΔHrxn(T)=ΔHrxn(298)+∫ΔCp dT 是 Kirchhoff 方程的直接應用；此例簡化為 ΔCp 取常數值（500K 處的瞬時值），適用於溫度區間不大的快速估算。
    • 巢狀字典儲存多種氣體的 Cp 係數，使得同一套計算邏輯可套用於不同物質——這是「數據驅動」程式設計的基礎思維。

    演算法步驟

    ⚙
    1. 建立巢狀字典儲存各氣體的 Cp 多項式係數
    2. 用 Horner 法計算 Cp(T) = a + T*(b + T*(c + T*d))
    3. 解析積分計算焓變化：ΔH = ∫T₂T₁ Cp dT
    4. 計算反應的 ΔCp = Σ(產物 Cp) - Σ(反應物 Cp)
    5. 用 Kirchhoff 近似修正反應焓：ΔHrxn(T) = ΔHrxn(298) + ΔCp×(T-298)

    題目

    氣體熱容多項式 Cp = a + bT + cT² + dT³ (單位：J/(mol·K), T 單位：K)

  N₂:  a=28.90, b=-1.571e-3, c=8.081e-6, d=-2.873e-9
  O₂:  a=25.48, b=1.520e-2, c=-7.155e-6, d=1.312e-9
  H₂O: a=32.24, b=1.924e-3, c=1.055e-5, d=-3.596e-9
  H₂:  a=29.11, b=-1.916e-3, c=4.003e-6, d=-8.704e-10

任務：
1. 用巢狀字典儲存 Cp 係數
2. 用 Horner 法計算各氣體在 T=500K 的 Cp
3. 計算各氣體從 300K 到 800K 的焓變化 ΔH = ∫Cp dT
4. 應用於 H₂ 燃燒反應：H₂ + 1/2 O₂ → H₂O

    逐步引導

    Step 1

    建立巢狀字典 cp_data，外層 key 為氣體名稱， 內層 key 為 'a', 'b', 'c', 'd'。

    提示：cp_data = {'N2': {'a': 28.90, 'b': -1.571e-3, 'c': 8.081e-6, 'd': -2.873e-9}, ...}
    Step 2

    用 Horner 法計算 T=500K 時各氣體的 Cp： Cp = a + T*(b + T*(c + T*d))

    提示：先取出係數，例如 a = cp_data['N2']['a']，再套入 Horner 公式。
    Step 3

    計算 300K → 800K 的焓變化（解析積分）： ΔH = a×(T2−T1) + b×(T2²−T1²)/2 + c×(T2³−T1³)/3 + d×(T2⁴−T1⁴)/4

    Step 4

    計算 H₂ 燃燒反應在 500K 的反應焓變化： ΔCp = Cp(H₂O) - Cp(H₂) - 0.5×Cp(O₂) ΔHrxn(500K) ≈ ΔHrxn(298K) + ΔCp×(500-298) 其中 ΔHrxn(298K) = -241818 J/mol（水蒸氣生成焓）

    提示：這是簡化計算，假設 ΔCp 在溫度區間內為常數（取 500K 的值）。

    💡
    重點整理：
    • Horner 法將多項式改寫為巢狀形式，減少乘法次數，提高計算效率和數值穩定性
    • 氣體熱容多項式 Cp(T) 是化工能量平衡計算的核心，常從 Perry's Handbook 查得
    • 焓變化 ΔH = ∫Cp dT 可對多項式解析積分，不需要數值積分
    • 反應焓隨溫度的修正：ΔHrxn(T) ≈ ΔHrxn(298K) + ΔCp×(T-298)
    🏭
    化工意涵：氫氣燃燒 H₂+½O₂→H₂O 是氫能與燃料電池產業的核心反應，準確的反應焓值是燃燒室設計、火焰溫度預測與熱回收系統效率評估的關鍵輸入。在化工能量平衡計算中，每一個反應器、熱交換器和分離塔都需要用 Cp 多項式計算物流的焓值變化；Perry's Chemical Engineers' Handbook 中收錄了數千種物質的 Cp 係數，工程師若能掌握 Horner 法與解析積分技巧，就能快速地在試算表或簡易程式中完成這些計算，而不必依賴大型模擬軟體。
